# 마하세븐 돌파 — 진입 시점 민감도 분석 결과

베이스라인: 시그널 다음날 시초가 시장가 매수 (n=261)

Exit 가격은 원본 그대로 → '진입 시점만 바꿨을 때의 효과' 추정.

Limit 시나리오: Low ≤ 한계가 충족 시만 체결, 아니면 거래 skip.

## 해석 가이드

- **expect/trade 가 높을수록 진입 가격이 낮았다 = 수익률 개선**

- **체결률(fill rate) 낮을수록 기회 손실 위험**

- **low (theoretical max)** 는 실제 도달 불가능한 ceiling — 분봉 데이터 + 정밀 매매로도 이 수준은 어려움

- limit 시나리오 fill rate 가 50% 미만이면 trade-off 부정적

- baseline 대비 +X% 개선 vs fill rate 감소를 종합해 채택 여부 결정
