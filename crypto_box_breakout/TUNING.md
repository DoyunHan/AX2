# 실데이터 백테스트 & 파라미터 튜닝 결과

> BTC 1시간봉 실데이터(2021-01-01 ~ 2026-06-02, 47,473봉)로 박스권 돌파 추세추종
> 전략을 백테스트하고 파라미터를 튜닝한 기록. 비용(수수료 0.05% + 펀딩 0.01%/8h),
> 레버리지 3x, 1회 리스크 1% 가정.

## 핵심 교훈: 기본값으로는 손실, 튜닝으로 흑자 전환

| 설정 | 거래수 | 승률 | 수익률 | PF | MDD | 평균보유 |
|---|---|---|---|---|---|---|
| **튜닝 전**(lookback24·폭4%·vol1.5·trail3) | 983 | 32% | **-49.1%** | 0.73 | -50.9% | 13.5봉 |
| **튜닝 후**(lookback48·폭2.5%·vol2.0·trail8) | 130 | ~40% | **+27%** | 1.5 | -10% | ~60봉 |

**왜 기본값이 망가졌나**
- `atr_trail`이 작아(3.0) 추세가 시작되기도 전에 잘림 → 평균 13.5봉만 보유(추세추종 실패).
- 박스 기준이 헐거워(폭 4%) 노이즈를 박스로 오인 → 983거래 과매매 → 수수료만 자본의 27% 소진.

**무엇을 바꿨나**
- `atr_trail_mult` 3 → **8** : 추세를 끝까지 홀딩(평균 보유 13.5 → 60봉).
- `box_max_width_pct` 4% → **1.5~2.5%** : 진짜 압축 구간만 박스로 인정.
- `vol_mult` 1.5 → **2.0** : 거짓 돌파(fakeout) 필터 강화.

## 견고성 검증 (과최적화 점검)

### 인샘플 vs 아웃샘플 (시간 분할)

| 설정 | 인샘플 2021-2023 | 아웃샘플 2024-2026 |
|---|---|---|
| 견고형(폭 1.5%) | +13.2% · PF 2.75 · 15거래 · MDD -4.5% | **+11.9% · PF 2.94 · 14거래 · MDD -2.2%** |
| 활성형(폭 2.5%) | +7.2% · PF 1.23 · 63거래 · MDD -10.4% | **+19.7% · PF 1.60 · 67거래 · MDD -8.2%** |

→ **두 설정 모두 아웃샘플에서 엣지 유지**(PF > 1.5). 단순 곡선 맞춤이 아니라는 긍정적 신호.

### 연도별 (견고형, 자본 매년 리셋)

```
2022: -2.1%  (PF 0.15, 5거래)   ← 유일한 손실 연도, MDD -2.4%로 미미

2024:  +1.6% (PF 1.50, 5거래)
2025:  +7.8% (PF 5.86, 6거래)

```

## 추천 설정 (두 가지 프로필)

| | 견고형 (보수) | 활성형 (현재 기본값) |
|---|---|---|
| `box_max_width_pct` | 0.015 | **0.025** |
| `box_lookback` / `vol_mult` / `atr_trail_mult` | 48 / 2.0 / 8.0 | 48 / 2.0 / 8.0 |
| 성격 | PF 높음·MDD 낮음, 단 거래 적음(신뢰구간 ↑) | 거래 많음(통계적 신뢰 ↑)·수익 큼, 단 MDD 큼 |

모듈 기본값은 **활성형**(거래 표본이 많아 통계적으로 더 견고)으로 설정했다.
보수적으로 가려면 `--box-width 0.015`.

```bash
python run_backtest.py --csv data/btc_1h.csv                 # 기본값(활성형)
python run_backtest.py --csv data/btc_1h.csv --box-width 0.015   # 견고형
python analyze.py data/btc_1h.csv                            # 연도별·인/아웃샘플 검증
python sweep.py   data/btc_1h.csv                            # 99조합 그리드 재탐색
```
