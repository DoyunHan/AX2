# 한국 주식시장 데이트레이딩 자동매매 시스템 구축 계획

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **목표** | 매일 아침 거래 시작 → 종가 전 모든 포지션 청산, 일일 소액 수익 달성 |
| **초기 자본** | 100만원 |
| **거래 시장** | KOSPI / KOSDAQ |
| **자동매매 API** | 키움증권 OpenAPI+ (Python) |
| **일일 수익 목표** | 0.5% ~ 2% (5,000원 ~ 20,000원) |
| **월 수익 목표** | 5% ~ 10% (50,000원 ~ 100,000원) |

## 2. 한국 주식시장 기본 구조

### 2.1 거래 시간

| 구분 | 시간 | 비고 |
|------|------|------|
| 장전 시간외 | 07:30 ~ 08:30 | 해외 시장 반영, 거래량 급증 종목 스캔 |
| 동시호가 | 08:30 ~ 09:00 | 호가 접수, 시초가 결정 |
| 정규장 | 09:00 ~ 15:30 | 핵심 거래 시간 |
| 시간외 종가매매 | 15:40 ~ 16:00 | 당일 종가로만 매매 |

### 2.2 거래 비용 구조 (2026년 기준)

| 항목 | 코스피 | 코스닥 |
|------|--------|--------|
| 증권거래세 (매도 시) | 0.05% + 농특세 0.15% = **0.20%** | **0.20%** |
| 위탁수수료 | 약 0.015% (키움 HTS 기준) | 약 0.015% |
| **1회 왕복 총비용** | **약 0.23%** | **약 0.23%** |

> 100만원 전액 1회 왕복 거래 시 약 2,300원 비용 발생
> 손익분기점: 매수가 대비 **+0.23% 이상** 상승해야 수익

### 2.3 주요 규정

- **가격제한폭**: ±30% (전일 종가 기준)
- **PDT 제한 없음**: 미국과 달리 한국은 당일매매 횟수 제한 없음
- **T+2 결제**: 매도 후 현금 입금까지 2영업일 소요
- **공매도**: 개인투자자 가능하나 조건 까다로움 (담보 105%, 사전교육 필수)

## 3. 매매 전략

### 3.1 종목 선정 기준

```
필수 조건:
├── 일평균 거래량 상위 (유동성 확보)
├── 일중 변동성 2% 이상 (수익 기회)
├── 시가총액 1,000억원 이상 (안정성)
└── 기술적 분석 지표가 명확하게 작동하는 종목
```

### 3.2 핵심 기술적 지표

#### (1) 이동평균선 (MA)
- **5일선 vs 20일선** (분봉 기준: 5MA vs 20MA)
- 골든크로스 (5MA > 20MA): 매수 신호
- 데드크로스 (5MA < 20MA): 매도 신호
- 정배열 확인: 단기 > 중기 > 장기 이동평균선

#### (2) VWAP (거래량 가중 평균가격)
- 가격 > VWAP: 상승 추세 확인 → 매수 유리
- 가격 < VWAP: 하락 추세 확인 → 매도 유리
- VWAP에서 크게 벗어난 경우 평균회귀 기대

#### (3) RSI (상대강도지수, 14기간)
- RSI > 70: 과매수 → 매도 신호
- RSI < 30: 과매도 → 매수 신호
- 다이버전스: 가격 신고가 vs RSI 하락 → 추세 반전 경고

#### (4) 볼린저 밴드 (20일, 표준편차 ×2)
- 하단 밴드 터치 + 반등: 매수 진입
- 상단 밴드 터치 + 반락: 매도 진입
- 밴드 수축(스퀴즈): 큰 변동 임박 → 돌파 방향 추종

#### (5) 거래량 분석
- 평균 거래량 대비 150% 이상: 거래량 급증
- 가격 상승 + 거래량 급증: 강한 매수세
- 가격 상승 + 거래량 감소: 추세 약화 경고

#### (6) 호가창 매수강도
- 매수강도(%) = 매수잔량 / (매수잔량 + 매도잔량) × 100
- 매수강도 > 55%: 매수 유리
- 매수강도 < 40%: 매도 유리

### 3.3 복합 매매 신호

#### 매수 진입 조건 (아래 6개 중 4개 이상 충족)

| # | 조건 | 수치 |
|---|------|------|
| 1 | 이동평균선 정배열 또는 골든크로스 | 5MA > 20MA |
| 2 | RSI 상승 전환 | 30 < RSI < 50 (과매도 회복) |
| 3 | 가격 > VWAP | Price > VWAP |
| 4 | 볼린저 밴드 중심선 위 | Price > BB Middle |
| 5 | 거래량 급증 + 양봉 | Vol > AvgVol × 1.2 |
| 6 | 호가창 매수 우위 | 매수강도 > 55% |

#### 매도(청산) 조건

| 유형 | 조건 | 수치 |
|------|------|------|
| **익절** | 목표 수익률 달성 | +2% ~ +4% |
| **손절** | 최대 허용 손실 | -1.5% ~ -2% |
| **신호 역전** | 매수 신호 → 매도 신호 전환 | 지표 역전 시 |
| **시간 기반** | 장 마감 30분 전 | 15:00 이후 청산 시작 |

### 3.4 시간대별 전략

| 시간대 | 전략 |
|--------|------|
| 09:00 ~ 09:30 | 갭 트레이딩, 오프닝 모멘텀 매매 |
| 10:00 ~ 11:00 | 추세 확인 후 추종 매매 (핵심 매매 시간) |
| 11:30 ~ 12:30 | 관망 (거래량 저하, 진입 자제) |
| 13:00 ~ 14:30 | 추세 재확인, 선별적 진입 |
| 14:30 ~ 15:00 | 포지션 정리 시작 |
| 15:00 ~ 15:30 | 모든 포지션 청산 완료 |

## 4. 리스크 관리

### 4.1 포지션 사이징

```
총 자본: 100만원
├── 1회 최대 투자: 20만 ~ 30만원 (전체의 20~30%)
├── 동시 보유 종목: 최대 3개
├── 1회 최대 허용 손실: 1만 ~ 2만원 (자본의 1~2%)
└── 일일 최대 투자 자본: 60만원 (자본의 60%)
```

### 4.2 일일 손실 한도

| 규칙 | 기준 | 조치 |
|------|------|------|
| 1회 손절 | -1.5% ~ -2% | 즉시 청산 |
| 연속 3회 손절 | 3연패 | 30분 휴식 후 재진입 여부 판단 |
| 일일 최대 손실 | -2만원 ~ -3만원 (자본의 2~3%) | **당일 거래 중단** |
| 주간 최대 손실 | -5만원 (자본의 5%) | 전략 재검토 |

### 4.3 손익비

- **목표 손익비**: 1:2 ~ 1:3
- 손절 1%에 대해 익절 2~3% 목표
- 승률 50%에서도 수익 가능한 구조 설계

## 5. 기술 구현 계획

### 5.1 개발 환경

| 항목 | 선택 |
|------|------|
| **API** | 키움증권 OpenAPI+ |
| **언어** | Python 3.9+ (32비트) |
| **GUI 프레임워크** | PyQt5 (OCX 연동 필수) |
| **래퍼 라이브러리** | pykiwoom 또는 직접 구현 |
| **데이터 분석** | pandas, numpy |
| **기술적 지표** | ta-lib 또는 직접 구현 |
| **OS** | Windows (OpenAPI+ 필수 요건) |

### 5.2 키움 OpenAPI+ 주요 함수

```
로그인:
├── CommConnect()          → 로그인 수행
└── GetLoginInfo()         → 계좌 정보 조회

시세 조회:
├── SetInputValue()        → TR 입력값 설정
├── CommRqData()           → 데이터 요청
├── GetCommData()          → 응답 데이터 추출
└── GetRepeatCnt()         → 반복 데이터 개수

실시간 데이터:
├── SetRealReg()           → 실시간 시세 등록
└── OnReceiveRealData()    → 실시간 데이터 수신 이벤트

주문:
├── SendOrder()            → 주문 전송 (초당 5회 제한)
└── OnReceiveTrData()      → 주문 결과 수신 이벤트

조건검색:
├── GetConditionLoad()     → 조건식 불러오기
├── SendCondition()        → 조건검색 실행
└── OnReceiveConditionVer() → 조건검색 결과 이벤트
```

### 5.3 API 제한사항

## 8. 주의사항 및 리스크 경고

### 현실적 인식

1. **높은 실패율**: 데이트레이딩 초보자의 90% 이상이 첫 3개월 내 손실
2. **거래 비용 부담**: 100만원 소액에서 왕복 0.23%의 비용은 큰 부담
3. **정보 비대칭**: 기관/외국인 대비 개인투자자의 정보 열위
4. **감정적 매매**: 손실 만회 심리가 가장 큰 적

### 성공을 위한 핵심 원칙

1. **규칙 준수**: 감정이 아닌 시스템이 매매 결정
2. **손절 기계적 실행**: 손절 기준 도달 시 무조건 실행
3. **일일 한도 준수**: 최대 손실 한도 초과 시 거래 중단
4. **매매 일지 작성**: 모든 거래 기록 및 복기
5. **첫 3~6개월은 학습 기간**: 수익보다 기술 습득에 집중
6. **복리 효과 활용**: 작은 수익을 꾸준히 쌓아가는 전략

### 초기 목표 재설정

```
1개월차: 시스템 안정화, 손실 최소화 (-5만원 이내)
2개월차: 손익분기점 달성 (수수료 포함 ±0)
3개월차: 월 1~3% 수익 달성
6개월차: 월 5% 수익 안정적 달성
12개월차: 자본 증액 및 전략 고도화
```
