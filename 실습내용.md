# 실습 내용 정리

## 프로젝트
**2025-2026 미국-이란 분쟁 인프라 피해 및 재건 비용 종합 보고서** HTML 페이지의 디자인 개선 및 데이터 시각화 작업

## 1. 디자인 분석 및 개선안 작성 (`design.md`)

기존 HTML 보고서의 디자인을 분석하고, 11개 항목의 구체적 개선안을 문서로 정리했다.

- 컬러 팔레트 리파인 (빨강 남용 문제 진단)
- 타이포그래피 고급화 (서체, weight, 자간)
- 레이아웃 & 여백 개선
- 카드 디자인 개선 (피해 등급별 분리, 히어로 수치 패턴)
- 데이터 시각화 추가 제안
- 마이크로 인터랙션 & 애니메이션
- 히어로 섹션 업그레이드
- 테이블 디자인 개선
- 네비게이션 개선
- 다크 모드 깊이감 (Elevation System)
- 인쇄 & 접근성
- 우선순위 표 (임팩트/난이도)

## 2. 디자인 개선안 전체 적용 (`us-iran-infrastructure-damage-report.html`)

`design.md`의 개선안을 HTML/CSS/JS에 전면 반영했다.

### 2-1. 컬러 & 톤
- 배경을 순수 블랙(`#0a0a0f`)에서 네이비 색조(`#0c0e1a`)로 변경
- 5단계 elevation 시스템 도입 (`--bg-0` ~ `--bg-4`)
- 빨강 액센트를 위험 데이터(비용, CRITICAL 뱃지)에만 한정
- 섹터별 고유 컬러 체계 적용 (에너지=앰버, 교통=틸, 군사=레드, 통신=바이올렛, 산업=골드, 수자원=시안)

### 2-2. 타이포그래피
- 본문 font-weight 300 → 400, `letter-spacing: -0.02em` 추가
- 숫자/통계에 `Cormorant Garamond` 세리프 서체 적용
- 섹션 넘버 크기 축소 (5rem → 3.5rem), 색상을 퍼플 계열로 변경
- h3 제목에 섹터별 컬러 도트 표시

### 2-3. 카드 디자인
- `critical` 카드 변형 + "CRITICAL" 뱃지 도입
- 카드 상단에 핵심 수치를 크게 표시하는 "히어로 수치" 패턴 (`card-hero-numbers`)
- 섹터별 좌측 보더 그라데이션 자동 적용
- 소규모 시설 카드는 2컬럼 그리드(`card-grid-2col`) 배치
- hover 시 `translateY` 제거, `box-shadow` 변경으로 교체

### 2-4. 데이터 시각화
- 재건 비용 CSS 수평 바 차트 (IntersectionObserver로 애니메이션 트리거)
- 통계 숫자 카운트업 애니메이션 (JS, `requestAnimationFrame` + easing)
- 네비게이션 하단 스크롤 진행률 표시줄

### 2-5. 인터랙션 & 애니메이션
- 스태거 딜레이 (`:nth-child`로 0.08s씩 순차 등장)
- 네비게이션 active 상태를 `.active` 클래스 토글 + CSS transition으로 변경
- 히어로 배경 미세 움직임 (`heroBgShift` keyframe)
- 스크롤 인디케이터를 텍스트에서 SVG 화살표로 교체

### 2-6. 레이아웃
- `content` max-width 1000px → 1100px
- 본문 텍스트 `max-width: 720px` 제한
- 섹션 간 여백 5rem → 7rem
- 섹션 구분자를 `border-bottom`에서 중앙 그라데이션 라인으로 교체
- 히어로 → 통계 구간에 그라데이션 전환(`.hero-fade`) 삽입

### 2-7. 테이블
- 행 간 줄무늬(stripe) 배경 추가
- 모바일 640px 이하에서 카드 형태 리플로우 (`data-label` 속성)
- 테이블 배경을 `--bg-2` 레벨로 통일

### 2-8. 접근성 & 인쇄
- `@media print` 스타일 추가 (배경 제거, URL 표시 등)
- `@media (prefers-reduced-motion: reduce)` 지원
- `--text-secondary` 색상 `#9b9b9b` → `#a8a8a8`로 대비 개선
- 테이블에 `scope="col"` 속성 추가

## 3. Mermaid 차트 5종 추가

Mermaid v11 CDN을 통해 다크 테마 커스텀 색상이 적용된 인터랙티브 차트 5개를 삽입했다.

| # | 섹션 | 차트 유형 | 내용 |
|---|------|-----------|------|
| 1 | 개요 | `flowchart LR` | 분쟁 구조 흐름도 (공격측 → 이란 피해 → 보복 → 걸프 6개국) |
| 2 | 에너지 | `timeline` | 2025.06 ~ 2026.04 주요 공격 연대기 |
| 3 | 재건 비용 | `pie` | 분야별 재건 비용 비중 파이 차트 |
| 4 | 재건 비용 | `flowchart TB` | 경제적 파급 효과 구조 (직접 피해 → 공급 충격 → 경제적 결과) |
| 5 | 전망 | `gantt` | 인프라 유형별 예상 재건 기간 로드맵 (2026~2032) |
